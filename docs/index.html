Instalação

npm install @led-telecom/design-tokens

Uso

JavaScript/TypeScript

import { designTokens, getToken } from '@led-telecom/design-tokens'; const primaryColor = designTokens.surface.primary; const errorColor = getToken('surface', 'error');

CSS

@import '@led-telecom/design-tokens/css/tokens.css'; .my-component { background-color: var(--led-surface-primary); color: var(--led-content-light); padding: var(--led-padding-m); }

API

designTokens

Objeto completo com todos os tokens de design.

getToken(category, key)

Obtém um token específico por categoria e chave.

getTokensByCategory(category)

Obtém todos os tokens de uma categoria específica.

getAllTokens()

Obtém todos os tokens de design.
