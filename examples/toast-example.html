Componente Toast - LED Telecom

Exemplos de uso do componente Toast baseado no design do Figma.

Estados do Toast

Error!

Que pena! Campo incorreto :(

sucess :D

Voce está fazendo tudo certo!

Atenção

Cuidado! Esqueceu de preencher o campo.

Controles Interativos

Uso com React

import { Toast } from '@led-telecom/components';

// Toast de erro
<Toast type="default" />

// Toast de sucesso
<Toast type="success" />

// Toast de warning
<Toast type="warning" />

// Toast customizado
<Toast 
    type="success" 
    title="Operação realizada!" 
    message="Os dados foram salvos com sucesso."
    onClose={() => console.log('Toast fechado')}
/>
