LED Telecom - Pagination Component

Componente de paginação seguindo o design system LED Telecom

Estados da Paginação

O componente Pagination suporta três estados principais conforme o design do Figma:

Estado Begin

Primeira página ativa com botão de próxima página destacado

...

Estado Middle

Página intermediária com botões de navegação anterior e próxima destacados

...

Estado End

Última página ativa com botão de página anterior destacado

...

Demonstração Interativa

Teste os diferentes estados da paginação:

Controles de Estado

Estado Atual: begin | Página: 1 | Total: 5 | Desabilitado: false

Características do Componente

🎨 Design System

Seguindo rigorosamente o design do Figma com tokens de design consistentes e acessibilidade.

📱 Responsivo

Adapta-se automaticamente a diferentes tamanhos de tela mantendo a usabilidade.

♿ Acessível

Suporte completo a navegação por teclado e leitores de tela.

⚡ Interativo

Estados de hover, focus e active com transições suaves.

🔧 Customizável

Props flexíveis para diferentes cenários de uso e integração.

🎯 TypeScript

Tipagem completa para melhor experiência de desenvolvimento.

Exemplo de Código

Como usar o componente Pagination em seu projeto:

import { Pagination } from '@led-telecom/design-tokens';

// Uso básico
<Pagination 
    property1="begin"
    currentPage={1}
    totalPages={5}
    onPageChange={(page) => console.log('Página:', page)}
/>

// Estado intermediário
<Pagination 
    property1="middle"
    currentPage={3}
    totalPages={10}
    onPageChange={handlePageChange}
/>

// Estado final
<Pagination 
    property1="end"
    currentPage={10}
    totalPages={10}
    disabled={false}
    className="custom-pagination"
/>
