Tag Component

Componente de tag de status do LED Telecom Design System

Variantes

O componente Tag possui três variantes principais:

Vencido
Vencido
Em Atraso
Em atraso
Pago
Pago

Estados

Diferentes estados do componente:

Default
Default
Hover
Hover
Pressed
Pressed

Tags Clicáveis

Tags que podem ser clicadas (com cursor pointer e efeitos hover):

Vencido
Clicável
Em Atraso
Clicável
Pago
Clicável

Uso em React

Exemplo de como usar o componente Tag em React:

import { Tag } from '@led-telecom/design-tokens'; // Tag básica <Tag label="Vencido" variant="Vencido" /> // Tag clicável <Tag label="Em Atraso" variant="Em atraso" clickable={true} onClick={() => console.log('Tag clicada!')} /> // Tag com estado específico <Tag label="Pago" variant="Pago" state="hover" />

Uso em HTML/CSS

Exemplo de como usar o componente Tag em HTML puro:

<!-- Tag básica --> <div class="led-tag led-tag--vencido"> <span class="led-tag__label">Vencido</span> </div> <!-- Tag clicável --> <button class="led-tag led-tag--em-atraso led-tag--clickable" onclick="handleClick()"> <span class="led-tag__label">Em Atraso</span> </button> <!-- Tag com estado hover --> <div class="led-tag led-tag--pago led-tag--hover"> <span class="led-tag__label">Pago</span> </div>

Props do Componente

interface TagProps { label: string; // Texto da tag variant?: 'Vencido' | 'Em atraso' | 'Pago'; // Variante da tag state?: 'default' | 'hover' | 'pressed'; // Estado da tag clickable?: boolean; // Se a tag é clicável onClick?: () => void; // Função de callback className?: string; // Classes CSS adicionais }
