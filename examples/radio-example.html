5. Uso com React/TypeScript

Exemplo de como usar o componente em React:

import { Radio } from '@led-telecom/design-system'; function MyForm() { const [selectedOption, setSelectedOption] = useState('option1'); return ( <div> <Radio label="Opção 1" state={selectedOption === 'option1' ? 'on' : 'off'} name="myGroup" value="option1" onChange={(checked) => checked && setSelectedOption('option1')} /> <Radio label="Opção 2" state={selectedOption === 'option2' ? 'on' : 'off'} name="myGroup" value="option2" onChange={(checked) => checked && setSelectedOption('option2')} /> </div> ); }

6. Características do Componente

  • Acessibilidade: Suporte completo a screen readers e navegação por teclado
  • Design System: Segue os tokens de design do LED Telecom
  • Responsivo: Adapta-se a diferentes tamanhos de tela
  • Estados: Suporte a estados on/off, hover, focus e disabled
  • Customizável: Permite classes CSS adicionais
  • TypeScript: Tipagem completa para melhor experiência de desenvolvimento
