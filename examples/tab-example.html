Como Usar

// Importar o componente import { Tab } from '@led-telecom/design-tokens'; // Uso básico <Tab label="Fatura" active={true} /> // Com estado desabilitado <Tab label="Relatórios" disabled={true} /> // Com callback de clique <Tab label="Configurações" onClick={() => console.log('Tab clicada!')} />

Props do Componente

  • label: Texto da tab (padrão: 'Fatura')
  • state: Estado da tab ('default' | 'disable')
  • active: Se a tab está ativa (boolean)
  • onClick: Função de callback para clique
  • disabled: Se a tab está desabilitada (boolean)
  • className: Classes CSS adicionais
